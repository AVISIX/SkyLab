Skylab Lua Editor is a dynamic ingame Lua Editor with beautiful Syntax Highlight and perfect ergonomic editing.

Current features include:
- Syntax Highlighting
- Undo & Redo
- Auto-Indenting
- Auto-Pair completion
- Pair detection
- Multi-word highlighting
- Find & Replace 
- Lua Compilation
